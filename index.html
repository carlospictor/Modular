Modular

Modular content loading with jQuery

Inspired by the article here: http://filamentgroup.com/lab/ajax_includes_modular_content/

Features

  • Lightweight - 585 bytes minified and gzipped
  • Data attributes - no need to touch the javascript once in place
  • Events - data can be loaded automatically, on click or viewport

Demo page showing click and viewport loading of content

Implementation

  1. Include jQuery in your page (ideally before the closing body tag) 
    
<script type="text/javascript" src="//ajax.googleapis.com/ajax/libs/jquery/1.8.2/jquery.min.js"></script>
  2. Get all the files from the /dist directory in the download
  3. Include the modular.js or modular.min.js file after jQuery.
    This example loads modular.js only if data-fragments are required on the page 
    
<script type="text/javascript">
	$(document).ready(function () {
			if ($('[data-fetch]').length > 0) {
                $.ajax({
                   	url: '../js/modular.js',
                    dataType: 'script',
                    cache: true,
                    success: function (data) {
                        $('[data-fetch]').each(function (index, value) {
                            $(this).modular();
                        });
                    },
                    error: function (XMLHttpRequest, textStatus, errorThrown) {
                        alert(textStatus);
                    }
                });
            }
							
		
	});


</script>
